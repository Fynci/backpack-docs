## Long titles

As this component is based on [Apple's Navigation Bar](https://developer.apple.com/design/human-interface-guidelines/ios/bars/navigation-bars/), it is not possible for the title to wrap onto multiple lines.

You must either make sure the title string you provide is short enough to fit on one line, or place a separate title label inside the screen.
